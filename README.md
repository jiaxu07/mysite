# MySite
MySite ver.0.1.0

# 安装

1. 将文件解压到服务器相应目录
2. 按照您的需求修改 units/mysconfig/config.yml
3. 运行根目录下的 com.php

# 更新

1. 新版本覆盖旧版本（保留配置文件 units/mysconfig/config.yml 和你原来的数据）
2. 修改配置文件 units/mysconfig/config.yml，将您的配置文件适配新版本。
3. 运行根目录下的 com.php
